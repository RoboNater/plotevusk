# DebugPlot Extension

Plot numeric variables during Python debugging.

## Features

This is a proof-of-concept extension that allows you to visualize numeric variables during Python debugging sessions in VS Code.

## Requirements

- VS Code 1.95.0 or higher
- Python debugger (debugpy)

## Extension Commands

This extension contributes the following commands:

* `DebugPlot: Plot Variable` - Plot a numeric variable from the current debug session
